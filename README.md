# GhostBot
Simple auto clicker for desktop. Fully controlled by keyboard shortcuts.

## To Build from source
Build to Windows executable file: `python -m PyInstaller  --icon="ghost-bot-icon.jpg" --onefile ghostBot.py`

## Notes
- If it isn't working in some windows, but works in others, try running it as admin.
- Has only been tested on Windows 10 and 11.
- Has English and Russian translation
